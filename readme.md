# Symulator obwodu ze sprzężeniem indukcyjnym

Projekt realizowany w ramach przedmiotu Metody Numeryczne. Aplikacja symuluje zachowanie obwodu elektrycznego z nieliniowym sprzężeniem indukcyjnym przy użyciu różnych metod numerycznych.

## Szybki start

### Wymagania wstępne
- Python 3.8 lub nowszy
- pip (menedżer pakietów Pythona)

### Instalacja krok po kroku

1. Pobierz projekt (lub sklonuj repozytorium):
   ```bash
   git clone https://github.com/filipdbr/symulator-obwodu-python.git
   cd symulator-obwodu-python
   ```
2. Stwórz i aktywuj środowisko wirtualne (zalecane, ale nie konieczne):
    ```bash
    # Windows
    python -m venv venv
    venv\Scripts\activate
    
    # Linux/Mac
    python3 -m venv venv
    source venv/bin/activate
    ```
3. Zainstaluj wymagane biblioteki:
     ```bash
    pip install -r requirements.txt
    ```
4. Gotowe
